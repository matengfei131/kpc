# 属性

## Header

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| fixed | 是否固定头部 | `Boolean` | `false` |
| theme | 主题颜色 | `"dark"` &#124; `"light"` &#124; `"white"` | `"dark"` |
| height | 高度 | `String` &#124; `Number` | `"64px"` |

## Aside

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| fixed | 是否固定侧边栏 | `Boolean` | `false` |
| collapse | 是否收起侧边栏 | `Boolean` | `false` |
| theme | 主题颜色 | `"dark"` &#124; `"light"` &#124; `"white"` | `"dark"` |
| width | 宽度 | `String` &#124; `Number` | `"200px"` |
